## Failed Approaches

- **Monkey-patch `VpnTray.build_menu` per aggiungere language submenu**: conflitti con refresh ciclico GTK, menù lingua non appariva. **Soluzione**: inline diretto in `build_menu()` con `set_lang()` method.
- **Bash heredoc `cat >>` per scrivere python file**: ha causato perdita di metodi (`toggle_country_lock`, `set_interval`, `_tick`, `apply_rotation`, `on_rotate`). **Soluzione**: Python inline `s.replace(old_block, new_block, 1)` con stringa precisa.
- **`nohup python3 script.py &`** per restart: spawnato processo doppio ogni volta. **Soluzione**: `setsid nohup python3 script.py </dev/null >log 2>&1 & disown` + cleanup esplicito dei PID precedenti.
